# Customer-Shopping-Behaviour-Analysis
Customer Behavior Dashboard
Overview

This project is a Power BI dashboard designed to analyze customer shopping behavior and purchasing trends.

The dashboard provides insights into:

Customer demographics
Purchase behavior
Product categories
Seasonal shopping trends
Subscription status
Review ratings
Shipping preferences
Tools & Technologies Used
Power BI
PostgreSQL
Python
Pandas
SQL
Features
Customer Analysis
Age group segmentation
Gender-based analysis
Customer subscription insights
Sales Analysis
Total purchase amount
Product category analysis
Seasonal trend analysis
Customer Preferences
Preferred shipping type
Color and size preferences
Product review ratings
Dataset Information

The dataset contains customer shopping behavior information such as:

Customer ID
Age
Gender
Item Purchased
Category
Purchase Amount
Location
Size
Color
Season
Review Rating
Subscription Status
Shipping Type
